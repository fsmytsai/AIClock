# Version Features
## 第一版包含以下功能：
1. 想賴床可延遲響鈴
2. 播報者 2 人
3. 可自訂背景音樂
4. 天氣播報及溫馨提醒
5. 播報 6 種新聞類別
6. 支持大字體手機不跑版
7. 可查看正在播報新聞
